# Skill Configuration Examples

This directory contains example configurations for various MCP skills.

## Available Examples

### 1. Basic Skill Configuration
- File: `basic-skill.json`
- Description: Minimal skill configuration with required fields

### 2. Advanced Skill with Environment Variables
- File: `advanced-skill.json`
- Description: Complex skill with multiple environment variables and parameters

### 3. API Integration Skill
- File: `api-integration.json`
- Description: Skill that integrates with external APIs

### 4. Database Skill
- File: `database-skill.json`
- Description: Skill that manages database operations

### 5. Multi-format Output Skill
- File: `multi-format-skill.json`
- Description: Skill that supports multiple output formats

## How to Use Examples

1. Copy the example file to the `skills/` directory
2. Modify the configuration to match your needs
3. Update the `name`, `description`, and other fields
4. Test your skill configuration

## Best Practices

- Use clear and descriptive names
- Include comprehensive descriptions
- Tag your skills appropriately
- Provide usage examples
- Document all parameters
- Include installation instructions
